# UI95 Editor Audit (updated 2026-05-02)

## Согласованное направление
- Совместимость со старыми форматами **не является приоритетом** на текущем этапе.
- Режимы `clean/debug` в экспорте **не нужны**.
- Базовый формат — **процентные метрики** для устойчивости при смене aspect ratio.
- Для каждого объекта допустим выбор типа вывода метрик:
  - `percent`
  - `absolute`

## Что улучшено в коде
- Экспорт остаётся «чистым», без служебного canvas-size в заголовке.
- На уровне узла добавлен параметр `metrics` в экспорт (`percent`/`absolute`).
- В Inspector добавлен выбор режима метрик для выделенного объекта.

## Дальше (приоритет)
1. Добавить `@format_version` и зафиксировать мини-спеку полей.
2. Ввести валидацию при импорте (range-check для pivot/anchor).
3. Вынести DSL-кодек отдельно от рендера/интеракций.
